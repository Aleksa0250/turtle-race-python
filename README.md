# 🐢 Turtle Race Game (Python)

A simple Python simulation game using the `turtle` module where multiple turtles race randomly until one reaches the finish line.

---

## 🚀 Features
- Choose number of racers (2–10)
- Random movement for each turtle
- Visual race using Python Turtle graphics
- Countdown before the race starts (3–2–1–GO)
- Finish line detection
- Winner displayed on screen and in console

---

## 🛠️ Technologies Used
- Python 🐍
- Turtle Graphics module
- Random module
- Time module

---

## 🎮 How to Run
1. Make sure you have Python installed
2. Run the script:
   ```bash
   python turtle_race.py
